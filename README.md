# Go Pokedex

The Go Pokedex is a REPL[^1] application that allows for the following commands to be run against the [PokeAPI](https://pokeapi.co/).

- `catch` - Catch a Pokemon by name.
- `exit` - Exit the application.
- `explore` - Display the Pokemon which can be found in a given location area by id or name.
- `inspect` - Inspect a caught Pokemon by name.
- `help` - Displays a help message.
- `pokedex` - Lists all caught Pokemon in your Pokedex.
- `map` - Displays the names of the next 20 location areas in the Pokemon world.
- `mapb` - Displays the names of the previous 20 location areas in the Pokemon world.

[^1]: REPL

    **R**ead - Read a command input  
    **E**valuate - Process the associated logic  
    **P**rint - Print the result  
    **L**oop - Circle back for more command input

    A terminal's command line is a REPL.
